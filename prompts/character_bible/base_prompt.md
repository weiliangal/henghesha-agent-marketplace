# Character Bible Prompt

Create a production-ready character bible entry for a US-localized cast member.

Required outputs:

- `character_id`
- `english_name`
- `age_range`
- `ethnicity_look_target`
- `body_type`
- `acting_energy`
- `speaking_rhythm`
- `wardrobe_profile`
- `makeup_hair_notes`
- visual reference checklist
- `must_keep`
- `must_avoid`
- scene fit notes

Rules:

- Keep the character stable across shots.
- Separate story-driven wardrobe changes from accidental generation drift.
- Optimize for repeatability in a multi-shot series.
