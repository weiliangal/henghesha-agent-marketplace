# Shot Generation Prompt

Generate a shot that preserves story function, emotional beat, blocking logic, and motion direction.

Inputs:

- shot summary
- scene bible
- character bible
- route recommendation
- reference performance notes

Outputs:

- a shot that can be cut into a short-drama timeline
- a clear camera and blocking interpretation
- a stable visual identity

Rules:

- Do not try to reproduce every pixel from the source shot.
- Prefer natural US short-form performance timing.
- Keep the result usable in editorial.
